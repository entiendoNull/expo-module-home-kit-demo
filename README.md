## Get started

1. Install dependencies

   ```bash
   bun install
   ```

2. Start the app

   ```bash
    bun expo prebuild --platform ios 
   ```

3. Build the project
   
   ```bash
    eas build --profile development --platform ios
   
    or (with connected device)
   
    bun run ios --device
    ```
